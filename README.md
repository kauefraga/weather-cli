<h4 align="center">A CLI that pretend to help you to find your wanted weather.</h4>

## ⬇️ How to download
```bash
git clone https://github.com/kauefraga/weather-cli.git
cd weather-cli
npm install
npm dev
```
Or downloading with yarn? `yarn && yarn dev`
